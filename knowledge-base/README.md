# Math Blast Pedagogy Knowledge Base

**Purpose:** This knowledge base supports content creators writing math stories for *Math Blast* — a space-adventure math magazine for Australian primary school kids (Years 1–6, ages 5–12).

Built to ensure every story, puzzle, and activity is grounded in evidence-based pedagogy, age-appropriate cognition, and genuinely engaging contexts.

---

## How to Use This Knowledge Base

### For Story Writers
1. Check **age-stage-guide.md** first — it tells you what vocabulary, story complexity, and problem types suit each year level.
2. Open the topic file (e.g. `fractions.md`) and go to the **Narrative Context Library** section for ready-to-use story hooks.
3. Use the **Common Misconceptions** section to write problems that gently address errors kids actually make.
4. Use the **Age-Stage Progression** section to calibrate difficulty and language.

### For Puzzle/Activity Designers
1. Read the **Core Pedagogy** section to understand the CPA (Concrete-Pictorial-Abstract) progression — stories should model this.
2. Check the **Innovative Approaches** section for game mechanics and formats with proven engagement.
3. Cross-reference **narrative-contexts.md** for a consolidated library of hook ideas across all topics.

### For Editors
- Each topic file flags common misconceptions. Avoid accidentally reinforcing them in story phrasing.
- The **age-stage-guide.md** gives sentence complexity, vocabulary tier, and emotional/motivational notes by year level.

## Australian Curriculum Alignment

All content aligns to the **Australian Curriculum: Mathematics** (v9.0) using year-level descriptors (Foundation to Year 6). Where relevant, content notes cross-strand connections (Measurement, Statistics, Algebra).

## Key Pedagogical Frameworks Referenced

- **CPA (Concrete-Pictorial-Abstract)** — Jerome Bruner's enactive-iconic-symbolic progression; widely adopted in Singapore Math and Australian Mastery approaches.
- **Mastery Teaching** — depth over pace; all students reach conceptual understanding before moving on.
- **Number Talks / Math Talks** — structured short discussions building number sense and reasoning.
- **Cognitive Load Theory** — Sweller; informing how we sequence information in stories.
- **Zones of Proximal Development (ZPD)** — Vygotsky; scaffolding challenge to just above current level.
- **Growth Mindset** — Dweck; language choices in stories matter for math identity.
- **ReSolve: Maths by Inquiry** — Australian ACARA/AAS funded program; problem-centred learning.
